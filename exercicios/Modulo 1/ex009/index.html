Listas Ordenadas - OL- Padrão

  1. Acordar
  2. Ligar para o joão
  3. Tomar café
  4. Escovar os dentes

Por padrão a lista começa no 1 e já é númerica, porém você pode mudar isto com alguns parametros type e start na tag ol, assim vc pode colocar como núemero romano ou por letras do alfabeto, exemplo abaixo:

  1. Acordar
  2. Ligar para o joão
  3. Tomar café
  4. Escovar os dentes

Aqui por exemplo ele esta utilizando o alfabeto para ordenar, e eu coloquei para começar pela quarta letra do alfabeto ou seja letra D.

Listas Não ordenadas - UL

  • Macarrão
  • Mantega
  • Chips
  • Cerveja

UL é uma lista não ordenada, que não precisa ser necessariamente enumerada, não precisa ter uma ordem etc... por padrão ela já vem com o disc, mas vc pode adicionar um circle ou square nestas listas, apenas estas 3 opções, disc,circle e square
